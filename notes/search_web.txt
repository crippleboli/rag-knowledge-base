2. 使用 streamableHttp 通信协议调用百炼平台的服务器上的工具
    使用方式:
    server = MCPServerStreamableHttp(
             name = "自定义 服务器的名字",
             params = {
                  mcp的链接信息
                  url : "mcp地址",
                  headers : {"Authorization" , f"Bearer {token}"},
                  timeout : 30
             }
             cache_tools_list=True,  是否缓存工具
             max_retry_attempts=3, 重试次数
     server 链接 .connect()
     server . 获取工具列表  list_tools()
     server . 调用工具信息  call_tool(工具名字:str,arguments:{})
     server . 断开链接内容  cleanup()
